# SAFAR - A Sharable itinerary
## Planned features
- Share itinerary using dynamic links.
- Add location to location paths.
- Add photos to location and paths.
- Add notes, lists, and disclamers about the locations visited.
## Tech stack
- React
- Tailwind
- Supabase
